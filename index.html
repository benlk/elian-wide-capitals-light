Download .woff .ttf .otf

To use this font, first download these font files and save them to your server. Then use a @font-face declaration in your CSS to load the files from your server. Modern browsers support WOFF, so you ought not need the TTF or OTF files, but they may be desired for other purposes. 


<style type="text/css">
	@font-face {
		font-family: ElianWideCapsLight;
		src: url('./ElianWideCapitalsLight.woff'),
			url('./ElianWideCapitalsLight.otf'),
			url('./ElianWideCapitalsLight.ttf')
		;
		font-weight: normal;
	}
	.ewcl {
		font-family: ElianWideCapsLight, sans-serif;
	}
</style>

Elian Wide Capitals Light is designed with the following ideas in mind:

Horizontal capitals
The 'long' segment of second- and third-cycle characters is aligned horizontally. In the case of characters with boxes, this is done by expanding the box.
Vertical lowercase letters
With the exceptions of the lowercase k, q, r, t, and z, where possible lowercase letters have been designed to use the 'long' segment of second- and third-cycle characters is aligned vertically.
Pure geometric design
Snap-to-grid, no serifs.

Because of the fineness of its lines, you should limit its use to high-contrast cases. In WCAG accessibility terms, the contrast level should be AAA at minimum.
